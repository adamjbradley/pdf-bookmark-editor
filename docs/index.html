PDF Bookmark Editor

Open a PDF, add or edit outline bookmarks, save a new copy — right in your browser.

⬇ Download extension Latest release zip · Chrome & Edge · Free · Open source How to install → 3 clicks, no store account needed while in dev-preview

Works on Chromium-based browsers: Chrome, Edge, Brave, Arc, Opera, Vivaldi. Requires Chrome 116+ or Edge 116+.

Install in 30 seconds

  1. Download & unzip Grab the latest release and unzip it into a folder you'll keep (e.g. ~/extensions/pdf-bookmark-editor/). Don't delete the folder — the browser reads from it on every launch.
  2. Open the extensions page In Chrome, visit chrome://extensions/. In Edge, visit edge://extensions/. Toggle Developer mode in the top-right.
  3. Load unpacked Click Load unpacked and pick the folder that contains manifest.json. Pin the extension to the toolbar → click the icon → Open editor → drop a PDF.

What it does

Add & edit outline bookmarks

Nested bookmarks, rename in place, jump to page, delete branches.

Bundled PDF.js viewer

Renders any PDF right inside the extension. Zoom, page nav, keyboard shortcuts.

Writes a real /Outlines dict

Bookmarks show up in every reader — Chrome, Edge, Safari, Preview, Acrobat, Foxit, mobile.

Local, always

No uploads, no analytics, no login. Just storage permission for zoom preference.

Privacy

Nothing is uploaded, ever. The extension only asks for the storage permission to remember your zoom level between sessions. No host permissions, no service worker, no network access. Full policy: PRIVACY.
